🚀 Stock Price Prediction System

A Stock Prediction System built using Spring Boot, FastAPI, and Machine Learning.

🔐 Features

✔ Secure Login (Spring Security)
✔ Animated Frontend UI
✔ REST API Integration
✔ FastAPI ML Prediction
✔ Real-time Stock Price Prediction
✔ Supports Multiple Stock Symbols (AAPL, TSLA, GOOGL, etc.)
